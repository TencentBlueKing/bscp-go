## Overview

bscp-go 是蓝鲸基础配置平台(BK-BSCP)提供的用于快速接入KV SDK，配置热更新，执行前后置脚本的官方命令行工具

## Features

- 配置文件拉取
- 配置文件watch
- 灰度发布
- 多服务支持
- 前后置脚本
